基本

初始化新仓库：git init

检查状态：git status

暂存：git add

提交：git commit
远程仓库

链接远程仓库：git remote add

上传到服务器：git push

克隆仓库：git clone

从服务器上拉取代码：git pull
分支

创建新分支：git branch

切换分支：git checkout

合并分支：git merge
深入

比对两个不同提交之间的差别：git log

回滚提交：git revert
